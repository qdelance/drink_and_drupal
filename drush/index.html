Les bases

drush help

Infos sur le système

drush status

Purge le cache

drush cc (D7) ou cr (D8)

Accéder à la BD

drush sql

Réinitialiser le mot de passe de l'admin...

drush upwd --password="givememypasswordback" admin

Mettre le site en maintenance

todo

Télécharger Drupal

drush dl

Installer le site

drush site-install

Gérer des paquets

drush dl, en, dis, up
